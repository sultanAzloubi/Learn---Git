# الأمر git log --oneline

> يقوم هذا الأمر بتلخيص  رسالة التسجيل او Git logs وتاريخ المشروع بحيث ان يجلب كل المعلومات في سطر واحد ومختصر, من الاشيائ التي يطبعها :

- رقم ال Hash 
- يطبع لك رسالة ال Commit

**طريقة كتابة الأمر :**

<div dir="ltr">

```
git log --oneline
```
